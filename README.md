# Habr Python web autotests 

Проект для обучения автоматизации веб-тестирования на языке Python

### Список тестов для Acceptance-тестирования

- Поиск
- Пагинация
- Логин
- Хедер
- Блоги
- Попапы незарегистрированного пользователя
- Тултипы на постах
- Страница поддержки
